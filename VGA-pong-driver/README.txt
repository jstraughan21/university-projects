--------------------------------------------------------------------------------

The VGA-driver project contains source and testbench files for a FPGA to VGA 
display. The result is a 2-player version of the classic game PONG.

-> src  : source files written in Verilog
-> test : testbench files written in Verilog

-> _iverilog_run.py : Python script to run the iverilog simulation
-> constraint.xdc   : Constraint file for the Basys3 FPGA
